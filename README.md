# WechatGroupManagingBot

## Dependencies

* Python 3.5.2

* Django 2.0.4

* BeautifulSoup 4.6.0

* itchat 1.3.10

## Run Server

```python
python manage.py runserver
```

## Run Managing Bot

```python
python client.py
```

### WeChat Command

* register
  @xxxx register https://leetcode.com/yyyyyy

* update url
  @xxxx update url https://leetcode.com/yyyyy

* daily report
  @xxxx day
